BACK WORK

FINDGEEK

BALSAMIQ / PROTOTYPE / PHOTOSHOP / PHP / MYSQL / JAVASCRIPT / BOOTSTRAP / MVC

This is a capstone project for web programming. Findgeek application is a social network for people want to share creative idea and work with each other on this idea.

VISIT THE WEBSITE

SCENARIO

For instance, a person called Vincent got an interesting idea, and he wanted to find someone discussing this idea. That person should know some web techiques, or some sales approaches. Thus, Vincent posted his idea on the website, specifying skill or position he wanted. On the other hand, applicants would find the projects they are interested in and apply for certain position they are good at. Vincent could talk with applicants who applied his project. Also, applicants might bring some optimization of this idea. Finally, Vincent would make a decision to pick up the right persons, which would be a team for this idea in the future.

IDEAS

Seeking innovative ideas and backing geeks

Engaging in a interesting project to be a geek
